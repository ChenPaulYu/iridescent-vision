# Iridescent Vision


### Run Locally
```bash
# install dependencies
$ npm install

# start the dev server
$ npm run dev

# create a production bundle
$ npm run build

# preview the production build locally
$ npm run preview

```

### Deploy

```bash
# deploy to github pages
$ npm run deploy
```
